# PowerToys-backup

[!([CC0](https://mirrors.creativecommons.org/presskit/buttons/88x31/svg/cc-zero.svg)](https://creativecommons.org/publicdomain/zero/1.0/)

A Template repository of PowerToys settings backup.

## Usage

### Simple

1. Download this repository as a zip file and extract it to a folder.
2. Double-Click `backup.bat` to backup your settings
3. Double-Click `deploy.bat` to deploy your backup

Note: You can put this folder in a cloud storage folder (OneDrive, Google Drive, etc.) to sync and keep your settings.

### For Git users

### Backup

1. Use this repository as a template for your own repository.
2. Run `backup.ps1` to backup your settings.
3. Now backup your settings to your own repository. Commit and push the changes to your repository.

### deploy

1. Clone your repository to your computer.
2. Run `deploy.ps1` to deploy your settings.
